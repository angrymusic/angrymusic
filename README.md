# 이민재

구조와 사용자 경험을 함께 설계하는 프론트엔드 개발자입니다.

React, Next.js, TypeScript를 중심으로 제품을 만들고 있습니다.  
운영 도구, 실시간 대시보드, 복잡한 데이터 UI를 더 빠르고 정확하게 사용할 수 있게 만드는 일에 관심이 많습니다.

## About

- 운영자용 UI, 어드민, 관제형 대시보드를 개발합니다.
- 실시간 상태 처리, 데이터 흐름 설계, 입력 검증 같은 문제를 자주 다룹니다.

- 단순 기능 구현보다 사용 흐름과 운영 효율을 함께 개선하는 일을 선호합니다.
